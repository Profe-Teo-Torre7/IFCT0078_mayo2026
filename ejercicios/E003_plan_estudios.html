Plan de Estudios de Desarrollo Web

El siguiente programa organiza los contenidos principales de un curso introductorio de desarrollo web.

Módulos del Curso

  • HTML
    • Estructura básica de documentos.
    • Etiquetas semánticas.
    • Enlaces y navegación.
  • CSS
    • Selectores.
    • Modelo de caja.
    • Diseño responsive.
  • JavaScript
    • Variables y funciones.
    • Eventos del navegador.
    • Manipulación del DOM.

Actividades

  1. Ejercicios individuales
    1. Crear una página personal.
    2. Construir un menú de navegación.
  2. Proyecto final
    1. Diseñar la estructura del sitio.
    2. Implementar las páginas principales.
    3. Publicar el proyecto.
