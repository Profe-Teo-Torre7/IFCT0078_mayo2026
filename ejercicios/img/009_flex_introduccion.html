1. ¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un sistema de diseño unidimensional de CSS que permite distribuir, alinear y organizar elementos dentro de un contenedor de manera flexible y eficiente.

Antes de Flexbox era común utilizar floats, inline-block o hacks complejos para crear diseños. Flexbox simplifica enormemente la alineación horizontal y vertical. 

.contenedor{
    display:flex;
}

Al aplicar display:flex, el elemento se convierte en un contenedor flex y todos sus hijos pasan a ser elementos flexibles.

2. Ejes principales de Flexbox

Flexbox trabaja con dos ejes:

  • Main Axis (Eje principal)
  • Cross Axis (Eje transversal)

Por defecto el eje principal es horizontal (fila). 

display:flex;
flex-direction:row;

7. gap

Añade separación entre elementos flex.

Es preferible al uso excesivo de márgenes porque mantiene una separación consistente.

14. Casos de uso reales

Navbar

nav{
    display:flex;
    justify-content:space-between;
    align-items:center;
}

Ideal para logo a la izquierda y menú a la derecha.

Menú horizontal

ul{
    display:flex;
    gap:20px;
}

Permite distribuir enlaces de navegación fácilmente.

Centrado perfecto

.container{
    display:flex;
    justify-content:center;
    align-items:center;
}

El método más utilizado para centrar contenido horizontal y verticalmente.

Tarjetas responsivas

.cards{
    display:flex;
    flex-wrap:wrap;
    gap:20px;
}

Las tarjetas se reorganizan automáticamente.

Sidebar + contenido

.layout{
    display:flex;
}

.sidebar{
    width:250px;
}

.content{
    flex-grow:1;
}

Patrón extremadamente utilizado en dashboards.

Galerías

.gallery{
    display:flex;
    flex-wrap:wrap;
}

Distribución automática de imágenes.

15. Flexbox vs Grid

Flexbox Grid
1 dimensión 2 dimensiones
Filas o columnas Filas y columnas simultáneamente
Excelente para componentes Excelente para layouts completos
Más simple Más potente

16. Buenas prácticas profesionales

  • Usar gap en lugar de márgenes cuando sea posible.
  • Evitar order para lógica importante de accesibilidad.
  • Combinar Flexbox y Grid según la necesidad.
  • Usar flex-grow para layouts fluidos.
  • Evitar anidar demasiados contenedores flex innecesariamente.
  • Probar siempre en móviles.
  • Comprender claramente el eje principal y transversal.

17. Resumen rápido

display:flex;
flex-direction:row;
justify-content:center;
align-items:center;
flex-wrap:wrap;

Dominar Flexbox es uno de los conocimientos más importantes para cualquier desarrollador frontend moderno. La mayoría de componentes de interfaces, navegaciones, formularios, paneles, tarjetas y dashboards utilizan Flexbox como base principal de distribución.
